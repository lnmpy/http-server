# http-server

a static http-server in golang

## Usage

```
$ http-server -h <ip> -p <port> <dir>
```

## Installation

```
$ go get github.com/lnmpy/http-server
```

or

```
$ git clone git@github.com:lnmpy/http-server.git
$ cd http-server && make install
```
